C Identifiers

•	Identifies refer to the names of the variables , functions and arrays or structures.

•	These are user defined names and consist of sequence of letters and digits.

•	Lower- and uppercase letters are treated as distinct.

•	Rules for Identifiers:

–	First character must be an alphabet(or underscore)

–	Must consist of only letters, digits and underscore

–	Only first 31 characters are significant

–	Cannot use keyword

–	Must not contain white space




Examples of Valid Identifiers

•	age

•	x

•	total_amount

•	sum_2

•	myVar

•	_tempValue

Examples of Invalid Identifiers

•	2total (starts with a digit)

•	int (reserved keyword)

•	my-variable (contains a hyphen)

•	first name (contains a space)
