Arithmetic Operators in C

An arithmetic operator performs mathematical operations such as addition, subtraction, multiplication, and division on numerical values (constants and variables).



Example of Arithmetic Operator in C

#include <stdio.h>

int main()
{
    int a = 10, b = 4, sum, sub, multi, div;

    sum = a + b; // addition
    printf("a + b is %d\n", sum);

    sub = a - b; // subtraction
    printf("a - b is %d\n", sub);

    multi = a * b; // multiplication
    printf("a * b is %d\n", multi);

    div = a / b; // division
    printf("a / b is %d\n", div);

    return 0;
}

Output

a + b is 14
a - b is 6
a * b is 40
a / b is 2
